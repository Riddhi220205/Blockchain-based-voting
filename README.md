# Blockchain Based Voting System (Ethereum)

## Overview
This project is a decentralized voting system implemented as a smart contract on the Ethereum Sepolia testnet.

## Features
- Admin-controlled election lifecycle
- Candidate registration
- One-person-one-vote enforcement
- Transparent vote counting
- Fully on-chain logic

## Smart Contract
- Network: Sepolia Testnet
- Language: Solidity
- Compiler: v0.8.x

## Deployed Contract
Contract Address:

Sepolia Etherscan:https://sepolia.etherscan.io/address/0x883bf9C871D6aE5254c4d7AD8bFb4F1Ae36D702e

## How to Interact
1. Add candidates (admin)
2. Start election
3. Vote using different accounts
4. End election
5. Read results on Etherscan
